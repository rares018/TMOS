# TMOS

## Introduction

TMOS is a lightweight os for most devices,it can run in both python3 and micropython.

## beginner help

TMOS is actually quite beginner friendly,you can see all of the commands with the "help" command.To exit,enter break.If you get stuck in an endless loop in our REPL,just use CTRL+C (KeyboardInterrupt).

## user-made apps

TMOS has the built-in command "execfile",meaning the user can make executable files that add any kind of programmable utilities in cpython/micropython.The user can also see and edit the os.
